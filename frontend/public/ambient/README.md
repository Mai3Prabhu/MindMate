# Ambient Sound Files for Box Breathing

This directory contains ambient sound files used in the Mindful Breaths Box Breathing feature.

## Required Files

Place the following audio files in this directory:

- `ocean.mp3` - Ocean waves sound for relaxation
- `wind.mp3` - Gentle wind sounds
- `hum.mp3` - Soft humming/drone sound
- `forest.mp3` - Forest ambience with birds and nature sounds

## File Requirements

- **Format**: MP3
- **Duration**: 2-5 minutes (will loop automatically)
- **Quality**: 128kbps or higher
- **Volume**: Normalized to prevent sudden loud sounds

## Recommended Sources

You can find suitable ambient sounds from:
- Freesound.org (Creative Commons licensed)
- YouTube Audio Library
- Zapsplat (with account)
- Record your own ambient sounds

## Usage

These files are automatically loaded by the MindfulBreathsModal component when users enable ambient sounds during their breathing sessions.

## File Size Considerations

Keep files under 5MB each for optimal loading performance. Consider using compressed MP3 files to reduce bandwidth usage.
